Тема: Веб-додаток меню ресторану з можливістю оцінки страв.

Мета: Написання стайту для ресторану, на якому можна буде переглядати страви та залишати про них відгуки. Додавання до сайту необхідних елементів для зручного користування. Додавання зручної системи данних для додавання та вилучення страв з переліку.

Об'єкт дослідження: Система відображення інформації про страви та система відгуків.

Предмет дослідження: Аналіз роботи існуючих онлайн-сервісів ресторанів.

Постановка задачі: Розробка веб-додатку, за допомогою якого користувач переглядає доступні для замовлення страви, ціни на них, та відгуки інших відвідувачів. Розробка системи відображення та вибору страв за критеріями користувача. Розробка системи данних для їх оновлення та актуалізації.

Бізнес логіка:

Користувач:

  • Реєстрація
  • Авторизація
  • Перегляд страв
  • Пошук страв
  • Вибір режиму відображення
  • Перегляд відгуків відвідувачів
Користувач (зареєстрований):
  • Оцінка замовленої страви
  • Написання відгуку
Адміністратор:
  • Додавання страв
  • Видалення страв
  • Редагування інформації про страви
  • Модерація відгуків
